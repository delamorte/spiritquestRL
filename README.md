# spiritquestRL
A roguelike/adventure game using Python 3 and BearLibTerminal.

Based on [Roguelike Tutorial Revised](http://rogueliketutorials.com/) and [jcerise's roguelike tutorial for Go](https://jeremyceri.se/roguelikes/).
